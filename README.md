## O Projeto

Este repositório contém a interface de apresentação do **Aquino.Bot**, um assistente educacional focado em acessibilidade. Como desenvolvedor, meu objetivo técnico com esta landing page foi criar uma aplicação funcional, que aliasse um design agradável com boas práticas de responsividade e usabilidade.

A ideia aqui é demonstrar de forma honesta e direta como construo interfaces limpas, focadas em resolver o problema e proporcionar uma boa experiência para quem usa.

## Destaques Práticos

A estrutura visual foi montada para mostrar que uma ferramenta de inclusão pode ter uma cara moderna e ser fácil de navegar.

* **Responsividade (Mobile-First):** O layout foi construído para se adaptar bem a diferentes telas, garantindo que o site funcione de forma fluida tanto no celular quanto no computador.
* **Dark Mode Confortável:** A paleta de cores escura foi escolhida não só pela estética, mas para garantir um bom contraste e tornar a leitura menos cansativa para o usuário, respeitando a proposta do bot.
* **Interface Limpa (Clean UI):** Tive o cuidado de organizar o espaçamento, a tipografia e a hierarquia das informações para que o usuário entenda o produto logo de cara e encontre os botões de ação facilmente.
* **Leveza e Desempenho:** Por ser uma página direta ao ponto e sem excessos no código, construída com HTML5 e Tailwind, o carregamento é rápido e consome poucos recursos.

## Stack Tecnológico

* **Core:** HTML5
* **Estilização:** Tailwind CSS
* **Deploy:** Vercel
